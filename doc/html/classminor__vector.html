template<class E, std::size_t N>
class minor_vector< E, N >

Nで指定した要素を除いた(E::Dimension - 1)次元ベクトル NがEの次元数より大きければコンパイルエラーとなる

-*- example -*- minor_vector<vector3, 0>([1, 2, 3]) = [2, 3] minor_vector<vector4, 1>([1, 2, 3, 4]) = [1, 3, 4]
